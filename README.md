# Kanban Board Task Management

This project was developed as part of my **learning journey** to master modern software engineering practices. My primary goals for this project were to implement and understand:

1. **MVC (Model-View-Controller) Architecture**: Learning how to separate data, business logic, and user interface to ensure the codebase is modular
2. **Object-Relational Mapping (ORM)**: Utilizing SQLAlchemy to interact with databases through Python objects
3. **Persistent Local Storage**: Understanding the implementation of SQLite as a portable, serverless local database for desktop environments.
4. **Modern UI/UX Development**: Exploring the CustomTkinter library to create an aesthetic, dark-themed, and responsive desktop interface.

## Tech Stack
- **UI**: CustomTkinter
- **ORM**: SQLAlchemy
- **Database**: SQLite
- **Architecture**: MVC (Model-View-Controller)

## Setup
1. `python3 -m venv venv`
2. `source venv/bin/activate`
3. `pip install sqlalchemy customtkinter`
4. `python3 src/main.py`
